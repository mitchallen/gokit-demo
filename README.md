# gokit-demo

## To run

```
go run main.go
```

## Example

Example of how this module downloads the gokit:

```
$ go get github.com/mitchallen/gokit@v0.3.0
```

```
go: downloading github.com/mitchallen/gokit v0.3.0
go get: upgraded github.com/mitchallen/gokit v0.2.0 => v0.3.0
```
